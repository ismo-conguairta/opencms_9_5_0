Um einen Benutzer für OpenCms zu erstellen oder zu bearbeiten stehen Ihnen für alle notwendigen Eingaben Eingabefelder zur Verfügung:

  1. Benutzername
  2. NachName
  3. Email
  4. Passwort
  5. Passwort - Bestätigung

Für optionale Informationen ist zuerst kein Eingabefeld sichtbar. Separate "plus" Schaltflächen erlauben das Hinzufügen von Eingabefeldern für diese zusätzlichen, nicht zwingend benötigten Informationen (wie z.B. "Beschreibung").

Bitte Beachten 

Wenn ein neuer Benutzer hinzugefügt wird, muss er anschliessend einer Gruppe zugewiesen werden. Ansonsten wird der neue Benutzer sich nicht an den Workplace anmelden können, da die Workplace bestimmte Gruppenzugehörigkeit von Ihren Benutzern verlangt, bevor ein Zugriff auf Ressourcen oder Funktionen stattgegeben wird.
