Mit Hilfe dieses Menüpunktes können Sie Dateien, Dokumente oder Bilder auf den OpenCms Server hochladen und so Ihren Lesern beispielsweise als Download anbieten. Es existieren zwei Möglichkeiten des Hochladens, die beide Ihre zweckbezogenen Vorteile haben:

  • Hochladen durch Ihren Browser
  • Hochladen mittels eines Java Applet

Welche Methode Sie zum Hochladen verwenden, konfigurieren Sie in Ihren Benutzereinstellungen.  

Diese Aktion kann entweder über eine separate Schaltfläche der Titelleiste oder über die Schaltfläche "Neu" mit der Option "Neue Datei hochladen" gestartet werden.
